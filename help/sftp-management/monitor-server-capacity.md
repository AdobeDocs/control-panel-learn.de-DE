Um die Subdomain-Verwaltung in [!UICONTROL Control Panel]aufzurufen, gehen Sie zu:

* Direkt über die URL: [https://experience.adobe.com/#/controlpanel/domain](https://experience.adobe.com/#/controlpanel/domain)
